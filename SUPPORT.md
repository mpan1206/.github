# Support

## How to Get Help

- **🐛 Bug Reports**: [Open an issue](https://github.com/mpan1206/.github/issues/new?template=bug_report.yml) using the bug report template
- **✨ Feature Requests**: [Open an issue](https://github.com/mpan1206/.github/issues/new?template=feature_request.yml) using the feature request template
- **💬 Questions & Discussions**: Start a [Discussion](https://github.com/mpan1206/.github/discussions)

## Before Asking for Help

1. Check the project's **README** for setup instructions and FAQs
2. Search **existing issues** to see if your question has been answered
3. Search **Discussions** for related topics

## Response Time

This is a personal open-source project maintained in my spare time. I'll do my best to respond promptly, but please be patient.
